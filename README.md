# Private Clinic Website 

A simple HTML/CSS project modeled after real private clinic website. First full static front-end project I built while learning web development.


## Features

- Desktop-first layout (planned mobile-first improvements in future projects)
- Clean and simple design
- Content generated with AI
- Royalty-free images
